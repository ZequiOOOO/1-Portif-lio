Site: Ser Életrica Soluções em Engenharia

Esse é um site desenvolvido com muito esforço e dedicação, embora seja um site simples para quem já tem um bom conhecimento nessa área de desevolvimentos com HTML, CSS e JS.

Os sites do tipo Landing Page têm como objetivo especifico: Apenas a visualização ou o contato direto.

Por exemplo: Os clientes/usuários podem acessar o site para visualiazar os serviços ou produtos. Ou seja, eles podem navegar pelo conteúdo e entrar em contato via links, formulários/emails, com integrações via API e Frameaworks.

Tipo de Projeto:

Landing Page Responsiva.

Tecnologias Usadas: HTML5, CSS, JavaScript e Frameworks.

EmailJS API (para receber e enviar e-mails com informações).

Funcionamento da Página:

Na primeira aba, o cliente pode conhecer a empresa, benefícios, duvidas, e acessar contatos integrados via links.

N segunda aba da página, o cliente pode visualiazar os projetos realizados e ver 6 etapas com uma breve explicação do desenvolvimento do projeto

Na terceira aba da pagina, o cliente pode preencher seu nome, e-mail, estado e cidade, além de calcular seu consumo kWh (cálculo feito via JS).

Ao enviar os dados, o cliente recebe imediatemente o resultado do consumo, enquanto o proprietario da empresa recebe essas informções por email para desenvolver um projeto personalizado(integração via API).

O cliente ao mesmo tempo recebe um e-mail com uma mensagem automática do proprietario da empresa(envio automatizado via API).

Pelo interesse no Front-end, o que mais me cativou.

Para a prática e ter a primeira experiência com desenvolvimentoWeb real.

Novos conhecimento em HTML e CSS, além do segundo contato com JavaScript.
